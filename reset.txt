--hard:
1. передвигает HEAD и (состояние репозитория) на коммит назад
2. очищает индекс.
3. удаляет все незакомиченные изменения
GOAL: Если нужно вернуться к предыдущему состоянию HEAD: git reflog master / git reset --hard ORIG_HEAD
reflog хранит состояния в течении 30 дней, которые доступны.

--soft:
1. передвигает HEAD на n коммит назад.
2. не трогает индекс.
3. сохраняет все незакомиченные изменения
GOAL: объединить несколько коммитов, если они одинаковые(относились к одной задаче)
откат состояния на n коммитов назад и переделать коммиты.
При объединении коммитов при выполнении команды
git commit -c / -C можно копировать описание коммита который исправляем, при этом
-с - откроет редактор для редактирования сообщения коммита.
-C - просто скопирует сообщение коммита.
Так же данные флаги берут данные об авторе коммита.
